# Cleaner-V1.0

Program napisany w C++, który usuwa pliki cache (%temp%), skanuje dysk i wykonuje skan anty-wirusowy.

## Wymagania:
- Windows 10/11
- Kompilator C++
- Biblioteka `nlohmann/json`, do czytania plików JSON.

## Uruchomienie
1. Zbuduj projekt
2. Przygotuj plik `config.json` według potrzeb
3. Uruchom `Cleaner V1.0.exe` w folderze Release
